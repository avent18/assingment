This is globalTrends API assignment

this is a small application made with using nodejs+express integrated with gitHub REST API (public api)


## Tech stack i have used 
-nodejs  ->runtime environment
-expressjs -> server creation
-axios  -> for fetching data from api


## public api i used 

- `GET /users/{username}/repos`
- `GET /repos/{owner}/{repo}`

## api end points
-localhost:3000/repos
            provide api query parameters
-localhost:3000/repos/userName/repoName

## to run
-git clone<repo-name>
-cd GLOBAL TRENDS
-npm install
-node src/index.js
